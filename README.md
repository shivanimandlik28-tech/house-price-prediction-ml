# House Price Prediction using Machine Learning

This project predicts house prices using Machine Learning.

Technologies Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Models Used:
- Linear Regression
- Support Vector Machine
- Random Forest

Evaluation Metric:
- Mean Absolute Percentage Error (MAPE)
